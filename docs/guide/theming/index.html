Theming

Overview

Peppercorn allows you to customize the color scheme for your project. It supports several color labels which are directly supported by several components. These labels are:

  • stable
  • primary
  • secondary
  • calm
  • energized
  • danger
  • royal
  • dark
  • light
  • white
  • black
  • gray
  • border
  • underlay
  • clear

Usage

The kit uses the color palette from Material Design. We have selected certain colors from the palette to be used in the default theme. The remaining colors from Material Design can be accessed using the colors property in the theme module.

Custom Theme

There really isn't much to using custom colors through your project. Implementing a theme is as simple as creating a new module somewhere within your project and importing it when needed.
