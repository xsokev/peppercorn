Guide

Overview

Peppercorn is a collection of modules that aim to help in the development of mobile applications using React Native.

There are a growing list of components (Container, ActionBar, Badge, Button, Card, FloatButton, Icon, List), decorators (Range, ScrollContainer) and helpers.

Installation

Peppercorn works directly with React Native. You should have React Native installed on your system and prepared for development.

Peppercorn depends on react-native-vector-icons in order for the icons to display.

Getting Started

Using Peppercorn is the same as using many other React modules. To use a particular component in your project you would import the component and call it in your render method.
