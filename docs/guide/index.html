    Guide

    Overview

    React Native UI Kit is a collection of modules that aim to help in the development of mobile applications using React Native.

    There are a growing list of components, decorators and helpers.

    Installation

    UI Kit works directly with React Native. You should have React Native installed on your system and prepared for development.

    UI Kit depends on react-native-vector-icons in order for the icons to display.

    npm install react-native-vector-icons --save
    npm install react-native-uikit --save

    Getting Started

    Using UI Kit is the same as using many other React modules. To use a particular component in your project you would import the component and call it in your render method.

    import {Button} from 'react-native-uikit'
    <Button onClick={() => return}>Click Me</Button>
