ActionItem

The ActionItem component is used to add items to the ActionBar. These items are typically used to perform actions in your application. E.g. back button, menu button, or settings button.

ActionItem is a themeable component.

Usage

Properties

Props Types Description
children
  • arrayOf(node)
  • node
 The custom content that can be added to an ActionItem. This can be a single component or several components.
label
  • shape
  • string
 The text to be displayed in the ActionItem. You can pass a string to display the text in a simple Text component. The color of the text will be determined by the theme of the ActionItem. You can also pass a shape that has the text and theme and they will be used to display the text in the color you choose. The shape accepts: text:string, theme:string.
icon
  • object
  • string
 The icon to be displayed in the ActionItem. You can pass a string to display the icon from the default icon font. See Icon for more info. You can also pass an object that can be used to provide more information on how to display the icon. The object can contain any or all of the following: icon:string, set:string, theme:string, size:number, style:object
action
  • func
 The action that will be performed when the ActionItem receives a touch event.
disabled
  • bool
 Indicator as to whether the ActionItem can act upon the touch event.
style
  • any
 A JSON object which can be used to override the default styles of this component.
